Returns an iterator over all of the sockets in the set.

This function returns an iterator which always starts at the element after the last one seen in a previous use of the iterator. Regardless of where it starts the iterator will traverse all of the sockets. This round-robin behavior is important for avoiding starvation when sending packets from sockets.
